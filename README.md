# Ndovela Nkululeko — Data Analyst · Data Specialist · Data Migration Portfolio

Five end-to-end, runnable projects demonstrating the full range of data
work: cleaning messy real-world data, building stakeholder-facing
dashboards, migrating a legacy system to the cloud with validated
integrity, running production-style ETL pipelines, and writing
non-trivial SQL.

Every project in this repo runs — clone it, run the scripts, and you get
the exact numbers shown in each README, not static screenshots of
someone else's output.

## Skills demonstrated across this portfolio

- **Data cleaning & quality**: duplicate detection, missing-value strategy, format normalization, automated quality thresholds
- **Analysis & visualization**: EDA, KPI definition, dashboard design
- **SQL**: joins, aggregation, CTEs, window functions, correlated subqueries
- **Data migration**: schema normalization, referential integrity, checksum/row-count validation, rollback planning
- **Data engineering**: ETL pipeline design, idempotent loads, orchestration (Airflow), logging and observability
- **Communication**: every project's README states the business problem and result in plain language, not just code

## Repo structure

Each project folder is self-contained: its own README (problem →
approach → results → what I'd do at scale), its own runnable scripts,
and its own data. Clone the repo and `cd` into any project folder to run
it independently — no shared setup required beyond the Python packages
listed in each project's README.
